Server Side Logging A logging function is implemented in the MPU for diagnostic purposes. For this purpose information on the status of the MPU as well as errors and warnings are logged in a non-volatile memory. The logging can be started, stopped and read out via the web browser. When logging is running, the entries are live streamed into the below textarea.
